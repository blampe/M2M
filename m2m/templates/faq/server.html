Who runs servers?

You do! You, being the students, as distinct from the school itself (which has its own servers, but with which M2M does not truck).

Anybody can run a server -- your neighbors might, your friends might -- if you're not careful, even YOU might be running a server right now.

But who, specifically, runs servers? Well, I run M2M (obviously). But a complete list of all the people who run servers, and which servers they run, is a bit beyond the scope of this site. Not to mention my interest.
